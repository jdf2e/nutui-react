## 背景

随着 NutUI 的用户越来越多，在 NutUI 的交流群里对 React 版本的呼声也越来越高，我们积极响应社区的期待，于 2021 年年中开始规划并启动 React 版的开发，经过近半年的开发和打磨，将在 2022 年元旦前隆重推出 NutUI-React 元旦版。
