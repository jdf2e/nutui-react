## 常见问题

### 找不到所需的语言包？

如果上方列表中没有你需要的语言，欢迎给我们提 Pull Request 来增加新的语言包，可以参考[维吾尔语的PR](https://github.com/jdf2e/nutui-react/pull/526)。
